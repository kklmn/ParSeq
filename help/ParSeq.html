ParSeq is a python software library for Parallel execution of Sequential data analysis

ParSeq implements a general analysis framework with a data model, plotter, cross-data analysis and tunable widget work space. It also sets a structure to implement particular analysis pipelines as relatively lightweight Python packages.

ParSeq is intended for synchrotron based techniques, first of all spectroscopy.
